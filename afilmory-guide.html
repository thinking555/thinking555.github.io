使用 afilmory 创建公开图片浏览库指南

访问官网：https://afilmory.art

步骤 1：创建你的空间

1. 打开官网并点击「创建我的空间」

2. 输入你想要的网址

3. 使用 Google 或 GitHub 登录

4. 创建后可通过该网址公开访问图片

步骤 2：添加存储源

1. 设置名称

2. 类型选择 GitHub 仓库

3. 填写 GitHub 用户名

4. 填写 GitHub 仓库名称

5. 填写 GitHub 访问令牌

步骤 3：创建 GitHub 仓库

创建仓库并设置为 Public（公开）

步骤 4：生成 GitHub Token

Settings → Developer Settings → Personal access tokens (classic)

用途备注：随便填写

到期时间：无期限

权限：repo

步骤 5：上传图片

上传后即可通过你的网址公开访问

示例：
https://mytupian.afilmory.art/
